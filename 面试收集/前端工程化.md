#### 2. 设计模式

- 单例模式

```
保证一个类仅有一个实例，并提供访问它的全局访问点

function Single(){

}
Single.getInstance=(function(){
	let instance=null;
	return function(){
		if(!instance){
			instance=new Single()
		}
		return instance;
	}
})()
```
